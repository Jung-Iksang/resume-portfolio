## 📌 프로젝트 개요

정적인 이력서 대신 **방문자가 직접 탐험하는 3D 공간**으로 나를 소개하는 포트폴리오입니다.
React Three Fiber로 3D 씬을 구성하고 GSAP로 인터랙션·애니메이션을 연출했으며,
Next.js 정적 배포로 GitHub Pages에 호스팅했습니다.

---

## 🛠 기술 스택

| 구분 | 기술 |
|------|------|
| **Framework** | Next.js 16, React 19 |
| **3D** | Three.js, React Three Fiber, Drei |
| **애니메이션** | GSAP |
| **상태관리** | Zustand |
| **Styling** | Tailwind CSS |
| **언어** | TypeScript |
| **배포** | GitHub Pages |

---

## 🚀 실행 방법

```bash
git clone https://github.com/Jung-Iksang/resume-portfolio.git
cd resume-portfolio/Iksang_resume
npm install
npm run dev        # http://localhost:3000
```
